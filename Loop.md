# Loop
### While loop
Form
```
while (condition) {
  statement
]
```
Continue to execute `statement` over and over again as long as the `condition` is `true`.

### For loop
Form
```
for (int i = 0; i < 20; i++) {
  statement
}
```
- `int i = 0` initialization of *counter*
- `i < 20` continue condition
- `i++` change in the counter
